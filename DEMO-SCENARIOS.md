## Level 5 — 전체 CLI 통합 시나리오

> unit-converter / idea-generator / note-taker / mini-crm / wikiflow 5개 전부 사용

```
미국 고객사 NovaTech 글로벌 온보딩해줘.

1. 본사까지 거리 3,500마일이 몇 km인지 확인해줘
2. CRM에 회사(NovaTech, IT) 등록하고, 담당자 Jason Lee(jason@novatech.io) 추가하고, 딜(NovaTech 파트너십, stage: lead) 생성해줘
3. 비즈니스 아이디어 3개 생성해서 제일 좋은 거 즐겨찾기에 저장해줘
4. 노트에 온보딩 요약 저장하고 태그는 crm, global 달아줘
5. 위키에 NovaTech 워크스페이스 만들고 Onboarding 폴더 생성 후 고객사 소개 문서 작성하고 퍼블리시해줘
```

```
Q2 스프린트 플래닝 전체 세팅해줘.

1. 코딩 아이디어 3개 생성해서 전부 즐겨찾기에 저장해줘
2. 아이디어 각각 CRM 태스크로 high priority 등록해줘
3. 서버실 온도 기준 22°C가 화씨로 몇 도인지 확인해줘
4. 노트에 스프린트 목표 저장하고 태그는 work, sprint 달아줘
5. 위키에 Q2-Sprint 워크스페이스 만들고 Planning 폴더 생성 후 스프린트 계획 문서 작성하고 버전 생성 후 퍼블리시해줘
```

```
내일 바이브코딩 워크샵 풀세팅해줘.

1. 강의실 온도 기준 72°F가 섭씨로 몇 도인지 확인해줘
```
